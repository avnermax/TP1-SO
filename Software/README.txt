############## README #############################################################################################

 Esse arquivo tem a intenção de tornar o mais agradavel possivel a interação do usuario com o programa, para isso basta
seguir os passos abaixo para então executar o programa. 
 Para execução do Programa seguimos as instruções da expecificação do trabalho disponibilizada via portal didatico:

Passos para execução e finalização:

1º Esse programa pode/deve ser executado via terminal, possivel em Sistemas Operacionais que seguem o padrão POSIX.
	Abra o terminal de comandos (Ctrl+Alt+T), e acesse a pasta onde o programa foi salvo.
	tambem é possivel abrir o terminal com auxilio do mouse/touchpad já dentro da pasta do arquivo
 
2º Com o terminal aberto, usaremos o comando "make",
	Exemplo:	#"usuario"~<"diretorio">$ make  
	O comando executa o Makefile que criara um arquivo executavel de nome: shellso.

3º Com o fim do 3º passo chega a vez de executar o programa para isso daremos o comando "./shellso" como nos exemplos:
	Exemplo 1:	#"usuario"~<"diretorio">$ .\shellso
		
		Nesse modelo o programa executa apartir de forma interativa executando instruções digitados pelo usuario.
	
	Exemplo 2:	#"usuario"~<"diretorio">$ .\shellso "arquivo".txt'
		
		Nesse modelo o programa executa uma ou mais instruções apartir de um arquivo alocado no mesmo diretorio 
		de execução do programa.
			OBS: O arquivo.txt  deve conter como ultima instrução o comando "fim".

4º Caso carregado de forma interativa, sempre que o inserido o comando "fim" o programa finaliza. Seguindo com a inserção
   de um arquivo deve respeitar os exemplos dados acima.



 EXTRA: O programa carrega apenas um arquivo por vez dessa forma só é possivel passar como parametro apenas um .txt ou 
	carregar de forma interativa.
